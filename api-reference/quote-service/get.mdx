---
title: 'Get Quote Request'
description: 'Fetch a quote request by ID with all active quotes and the current best quote.'
---

```
GET /v1/quote-requests/{id}
```

Returns the full quote request object along with all active quotes for the current version and the computed best quote.

**Auth:** `Authorization: Bearer <privy_jwt>`

## Path Parameters

| Parameter | Type | Description |
| --- | --- | --- |
| `id` | string | Quote request ID. |

## Errors

| Code | Status | Description |
| --- | --- | --- |
| `UNAUTHORIZED` | 401 | Bearer token is missing, expired, or invalid. |
| `FORBIDDEN` | 403 | Authenticated user does not own this quote request. |
| `NOT_FOUND` | 404 | Quote request not found. |
