---
title: 'List Markets'
description: 'List active prediction markets on Totalis, grouped by event, across Kalshi and Polymarket. Filter by category, subcategory, frequency, date window, or full text search.'
openapi: 'GET /markets'
---

<Note>
  **Building parlays?** Every market carries an `exclusion_keys` array. Two
  markets cannot combine in the same parlay **if (and only if) their
  `exclusion_keys` intersect** — a pure set intersection, no correlation
  tables to maintain client side. The keys encode every rule: cross series
  groups, threshold ladders, same game sports correlation, Polymarket sub
  events. See `exclusion_keys` in the response schema below.
</Note>
